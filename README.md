  # Nightwatch Boilerplate
  Nightwatch end-to-end test automation boilerplate, using Commonjs and Expect.
  ## Getting Started
  	 git clone git@github.com:e2e-boilerplate/nightwatch-commonjs-expect.git
    	 cd nightwatch-commonjs-expect
    	 npm install
    	 npm run test
